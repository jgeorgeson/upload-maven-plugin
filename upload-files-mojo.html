upload:upload-files

Full name:

net.lopht.maven-plugins:upload-maven-plugin:0.6.0:upload-files

Description:

Uploads multiple files to remote repository.

Attributes:

  • Requires a Maven project to be executed.
  • Binds by default to the lifecycle phase: deploy.
