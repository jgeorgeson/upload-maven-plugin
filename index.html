upload-maven-plugin

A maven plugin for direct HTTP PUT upload of a file/folder to a specific server path.

Goals Overview

General information about the goals.

  • upload:upload-file Upload a single file.
  • upload:upload-files Scan a folder and upload contents.
  • upload:help Displays help for the plugin.
